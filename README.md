# 💼 Personal Portfolio Website

A modern, responsive, and interactive developer portfolio website built using HTML, CSS, and JavaScript. This project showcases skills, projects, and contact information with engaging UI animations and demos.

---

## 🚀 Features

- Clean and modern UI design  
- Smooth animations and transitions  
- Interactive particle background (Canvas API)  
- Typewriter text effect  
- Animated counters and skill bars  
- Fully responsive design  
- Interactive project demos:
  - CSS Animation Demo
  - Sorting Algorithm Visualizer
  - Pixel Drawing Canvas
  - Theme Toggle Engine
- Contact form with UI feedback

---

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3, JavaScript (Vanilla)  
- Styling: Custom CSS with variables  
- Graphics & Animation: Canvas API, CSS Animations  
- Interactivity: DOM Manipulation, Intersection Observer API

   cd portfolio  

3. Open index.html in your browser (or use Live Server)
